# VRC Blend Shape Clamper

[English](README.md) | [简体中文](README_zh.md) | 日本語

ブレンドシェイプを非破壊でクランプ（制限）するための NDMF プラグインです。

主に、表情用のブレンドシェイプを調整した後に、他のブレンドシェイプによって発生する貫通（クリッピング）を防ぐ目的で使用されます。

[更新履歴 & 説明書](https://product-instructions.luiscreamed.com/?i=blend-shape-clamper)

## 機能

- 単一のコンポーネントで、ブレンドシェイプを非破壊でクランプできます。
- 設定時にブレンドシェイプのプレビューが可能です。
- アニメーションクリップ内のブレンドシェイプ値もあわせて処理します。
